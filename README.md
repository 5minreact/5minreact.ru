## Инструкции по сборке

```
npm install
```

**Режим разработки** (локальная версия будет доступна по localhost:8000):
```
gatsby develop
```

**Сборка и публикация**:
```
gatsby build
npm install -g gh-pages
gh-pages -d public
```

Либо одной командой:
```
npm run deploy
```
